# Samsung árfigyelő
Samsung mobiltelefon árfigyelő és naplózó

###############################################################

A www.ses.hu oldal megnyitását követően elfogadjuk a cookie-kat, majd bejelentkezünk.
Bejelentkezést követően ellenőrizzük a bejelentkezés tényét, vagyis megnézzük hogy a user neve látszik-e az oldalon.
Ezt követően a mobil szekcióból kiválasztjuk a Galaxy S szériát.
Azon belül az S26 Ultra alpontot.
Szín esetében a feketét választjuk, tárhely esetében pedig az 512 Gb-os verziót.
Az oldal egyetlen terméket ad vissza, melynek megnyitjuk a részletes oldalát is.
Az itt szereplő 3 árat (eredeti ár/kedvezményes ár/törzsvásárlói ár) kimentjük külön változókba.
Az így kinyert árakat pedig elmentjük az adott napi lekérdezés sorába egy csv file-ba.
Továbbá egyáb matematikai műveletekkel megvizsgáljuk hogy forintosítva, valamint százalékos értékben kifejezve mekkora kedvezmény érhető el a teljes árhoz képest.
